# 🛡️ Mini SOC - Security Operations Center

A lightweight Security Operations Center (SOC) simulation system for learning and demonstrating cybersecurity monitoring concepts.

![SOC Dashboard](https://img.shields.io/badge/SOC-Monitoring-blue)
![Python](https://img.shields.io/badge/Python-3.6+-green)
![License](https://img.shields.io/badge/License-MIT-yellow)

## 📊 Overview

Mini SOC is a complete security monitoring system that simulates real-world SOC operations:
- **Log Generation**: Creates realistic security events
- **Threat Detection**: Real-time pattern-based alerting
- **Web Dashboard**: Live visualization of security events
- **Alert Management**: Severity-based classification

## ✨ Features

- 🔴 Real-time security event monitoring
- 🟡 Pattern-based threat detection (Brute Force, SQL Injection, Malware)
- 🟢 Severity classification (Critical, High, Medium, Low)
- 📊 Live web dashboard with auto-refresh
- 📝 Comprehensive logging and alerting
- 🎨 Color-coded console output
- 📈 Statistical reporting
- 🖥️ Windows compatible
