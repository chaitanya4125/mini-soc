A lightweight Security Operations Center (SOC) simulation system for learning and demonstrating cybersecurity monitoring concepts. Built with pure Python - **zero external dependencies!**

## 📊 Overview

Mini SOC is a complete security monitoring system that simulates real-world SOC operations. It generates realistic security events, detects threats in real-time using pattern matching, and provides a beautiful web dashboard for visualization.

### 🎯 Perfect For:
- **Cybersecurity Students** - Learn SOC operations hands-on
- **Job Seekers** - Impressive portfolio project for security roles
- **Instructors** - Demonstrate security monitoring concepts
- **Developers** - Understand threat detection systems

## ✨ Features

### 🔴 Threat Detection
- Real-time security event monitoring
- Pattern-based detection engine
- 6 pre-configured detection rules
- Configurable thresholds and time windows
- Severity classification (Critical, High, Medium, Low)

### 📊 Dashboard
- Live updating web interface
- Real-time statistics and counters
- Color-coded alert visualization
- Streaming event log viewer
- Auto-refresh every 3 seconds
- Connection status indicator

### 🛠️ Technical
- Pure Python - No pip install needed!
- REST API endpoints
- JSON-based configuration
- Console color output
- Cross-platform compatibility
- Comprehensive logging
